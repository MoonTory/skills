# System

Internal vault infrastructure. Templates, conventions, and attachments live here.

## Contents
- **Templates/** — Note templates using Templater syntax (`<% tp.* %>`)
- **Conventions/** — Standards for frontmatter, linking, and tags
- **Attachments/** — Images, PDFs, and other media referenced by notes

## Maintenance
- When adding a new template, place it in `Templates/` and document any required frontmatter fields.
- Keep conventions up to date as your workflow evolves.
