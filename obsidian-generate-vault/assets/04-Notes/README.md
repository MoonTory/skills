# Notes

Permanent, atomic notes — each one captures a single idea in your own words. This is your long-term knowledge base.

## Workflow
1. Create a note using the **Atomic Note** or **Technical Note** template.
2. Write the idea in your own words — not just a copy-paste.
3. Link to related notes liberally. Connections are the point.
4. Add tags from the conventions in `99-System/Conventions/Tags`.

## Principles
- **One idea per note** — if it covers two topics, split it.
- **Evergreen** — update notes as your understanding evolves.
- **Linked** — a note with no links is an orphan. Connect it.
