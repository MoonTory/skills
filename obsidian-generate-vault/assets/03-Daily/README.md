# Daily Notes

One note per day using the **Daily Note** template. Serves as your journal, task list, and daily log.

## Workflow
1. Create today's note from the Daily Note template (or use the Calendar plugin).
2. Plan your day in the morning.
3. Capture notes, tasks, and thoughts throughout the day.
4. Review at end of day — move permanent ideas to `04-Notes`.

## Tip
Configure the **Templater** plugin so new daily notes automatically use the `99-System/Templates/Daily Note.md` template.
