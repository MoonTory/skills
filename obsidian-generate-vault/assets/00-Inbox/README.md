# Inbox

Unprocessed captures go here. This is the default landing spot for quick thoughts, clippings, and anything you haven't sorted yet.

## Workflow
1. Capture the idea — don't worry about formatting.
2. During a weekly review, move each note to the right folder (`01-Projects`, `02-Areas`, or `04-Notes`).
3. If it's not worth keeping, delete it.

The goal is to keep this folder **close to empty**.
