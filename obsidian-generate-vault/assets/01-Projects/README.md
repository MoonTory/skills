# Projects

Active efforts with a clear goal and deadline. Each project gets its own subfolder or note.

## Workflow
1. Create a new note using the **Project Brief** template.
2. Link related meeting notes, RFCs, and daily entries back here.
3. When complete, change the status tag to `#status/done` or `#status/archived`.

## Examples
- Ship feature X
- Write blog post on Y
- Prepare conference talk
